Testowane na Python 3.11.6

Instalacja wymaganych bibliotek:
pip install tensorflow numpy pillow matplotlib scipy

Uruchamianie programu za pomocą wiersza poleceń
Uruchamiamy CMD
cd 'sciezka do programu'
python model.py
python app.py


Najpierw uruchamiamy model.py, aby stworzyć i wytrenować model sieci neuronowej.
Materiały do trenowania sieci znajdują się w folderze images.
Następnie uruchamiamy app.py.
Program wczyta 10 losowych zdjęć ptaków z fodleru test_img i określi ich gatunek.
